# Deploy to Golem Cloud

Golem Cloud is hosted version of Golem. It is easiest and fastest way to run Golem [agents](/v1.5/concepts/agents),
in the cloud, at scale, without any infrastructure setup or maintenance required.

Golem Cloud can be managed by:

- [golem-cloud-cli](/v1.5/cli)
- [REST API](/v1.5/rest-api)
